README;
========================================================
-A megaman level from scratch using a game engine I've built

-Run 'Game.exe' to play.

-Controls:
	  right = rightArrow
	  left = leftArrow
	  shoot = z
	  jump = x
	  pause/unpause = p/o
	  debug/nodebug = vb/c

- n.b Made on Windows 10. I have not tested on linux although it should still run

made with python 3.7.3 and pygame 1.9.6
